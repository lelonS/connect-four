# connect-four

## Testing

### Install dependencies
```bash
npm install
```


### Coginitive Complexity

```bash
npm run complexity
```

### Jest (unit tests)

```bash
npm run test-jest
```

### Cypress (ui tests)

1. Start a test server
```bash
npx http-server . -p 5500
```

2. Run Cypress as an UI or in terminal

```bash
npm run test-cypress-ui
```

**or**

```bash
npm run test-cypress
```
